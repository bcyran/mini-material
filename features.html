Tooltips

There are also tooltips. You can use them in four positions:

  • tooltip--top
  • tooltip--right
  • tooltip--bottom
  • tooltip--left

Tooltips doesn't use any JS, only CSS!

Colors

Color Palette includes 21 colors which are base colors from Google Material Design guidelines: red, pink, purple, deep purple, indigo, blue, light blue, cyan, teal, green, light green, lime, yellow, amber, orange, deep orange, brown, grey, blue grey, black and white.

Media Queries

Mini Material has variety of classes for hiding content depending on screen size. This system relies on four screen sizes: s, m, l, xl defined by breakpoints variables. You can use --up and --down modifiers if you want to target more than one size.

Resize browser window to see how list of examples is changing according to size:

  • hide--s
  • hide--m--down
  • hide--m--up
  • hide--l--down
  • hide--l

SASS Variables

If you are using SASS you can adjust many aspects of library using variables. For example:

  • transition-duration
  • button-height
  • page-content-width
  • card-image-height
  • page-font-size
  • footer-height

And much more!
